# ページスニペット

## Overview

指定したURLプレフィックス単位でスニペットを管理します。
例えば、「http://hoge」と入力した場合、「http://hoge」から始まる全てのURLで使用できます。

## Installation

1. chromeで「 chrome://extensions/ 」を開きます。
1. pageSnippet.crxをドラッグ&ドロップしてください。

## Caution

* 「%%%」という文字列をデータ管理上の区切り文字として使用しているため、スニペットに入力することはできません。
